About

Welcome to HoneydewLimeade.com

This is a personal site built from the bottom up, incorporating the building blocks of web development as I go.

Why HoneydewLimeade? It’s a nod to my career in the hospitality industry where I specialized in cocktails. Honeydew Limeade is one of the drinks I have published in Batch Cocktails by Maggie Hoffman.

Working professionally in the hospitality industry for over 15 years, I honed skills of craftsmanship, discipline, and creativity. With a degree in computer science, I’m now utilizing those skills and applying them to web development.

This site is written in Bootstrap 5 framework, with inspiration from Bootstrapmade templates. General references for HTML, CSS and JavaScript were primarily sourced from w3schools. The code is publicly available on Github.
